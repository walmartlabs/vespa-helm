Vespa Helm Charts Repository

Official Helm charts for deploying Vespa to Kubernetes

🚀 Quick Start

Add the Helm Repository

helm repo add vespa https://walmartlabs.github.io/vespa-helm
helm repo update

Install Vespa

# Install with default values
helm install my-vespa vespa/vespa

# Install with custom values
helm install my-vespa vespa/vespa -f my-values.yaml

📋 Available Charts

  • vespa - Complete Vespa deployment with all components

📚 Documentation

For detailed installation instructions, configuration options, and examples, visit the GitHub repository.

🔧 Prerequisites

  • Kubernetes cluster
  • Helm 3.x
  • Istio service mesh

📝 Configuration

The chart supports extensive configuration through values.yaml. Key configuration areas include:

  • Vespa schemas and indexing
  • Resource allocation and scaling
  • Istio integration and ingress
  • Persistent storage configuration
  • Monitoring and metrics
